# People

Team directory. Claude maintains this file.

Each person has:

- **Role** — PI / postdoc / PhD student / lab tech / collaborator / external
- **Projects** — which projects they're involved in
- **Cadence** — 1:1 frequency, last contact date
- **Owes me** — what they committed, when, status
- **I owe** — what you committed to them, when, status

---

## Template

```
### <Name>

- **Role**: postdoc
- **Projects**: [project-id-1, project-id-2]
- **Cadence**: bi-weekly 1:1, last contact YYYY-MM-DD
- **Owes me**:
  - [ ] <item> — committed YYYY-MM-DD, due YYYY-MM-DD
- **I owe**:
  - [ ] <item> — committed YYYY-MM-DD, due YYYY-MM-DD
- **Notes**: strengths, interests, career stage, preferred comm style
```
