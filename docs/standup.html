What fires it

SessionStart hook in .claude/settings.json. When you open Claude Code in this repo, it checks whether /standup has run today. If not, it runs. 

{
  "hooks": {
    "SessionStart": [{
      "matcher": "*",
      "command": ".claude/scripts/maybe-standup.sh"
    }]
  }
}

Manual invocation

Run it any time in Claude chat: 

/standup

Backed by .claude/commands/standup.md — sets the exact structure Claude should produce.

Everything above comes from markdown in this repo. No database, no API key.
